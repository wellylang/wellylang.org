Let's try some examples

At this stage you might have already installed Welly and opened your terminal, let's start programing, You can copy the code snippets to try on your terminal

Hello World! 

>>>> var three = 3
>>>> three = three+3 
>>>> three

6: int

In the above example, when you input "Hello World!" on terminal it returns "Hello World!" along with the type which in this case is Str (string). When assigning strings to variables or constants its important to use double quotes. You can also store the value in a variable.

Variables

>>>> var three = 3
>>>> three = three+3 
>>>> three

6: int

var keyword is used to store value of three(variable's name) and = operator is used to assign the value 3. You can perform any changes to the value, as in example three+3 will add 3 to value and display 6 . Read more about variables here

String Concatenation

>>>> var  firstName = "Dave"; var lastName = "Smith"
>>>> firstName + "  " + lastName

"Dave Smith": str

String Concatenation can be performed in Welly using the standard Concatenation procedure. + operator is used to combine two strings

Constants

>>>> const x = 3
>>>> x = x + 3

ConstnessVMError: can only assign to "var" locations

>>>> x + x
6: int

>>>> x

3: int

>>>> var y = x - x
>>>> y

0: int

const keyword is used to declare a constant followed by the constant's name x same rules are applied to declare the name of constant as variables. = operator is used to store the value 3 . Constant's value can not be changed. As you can see the error while trying to change the value. You can perform any operation on constants, but that does not affect the value of constant as in above example, even after the addition operation is performed on x original value stays unchanged. read more about constants here
